# Blood-Dona-Pred
Blood Donation Prediction using Machine Learning | EDA, Data Preprocessing, Feature Scaling, and Classification models (Logistic Regression, Decision Tree, Random Forest, Naive Bayes, SVM) to predict repeat blood donors.
🩸 Blood Donation Prediction using Machine Learning

This project predicts whether a blood donor is likely to donate blood again based on historical donation records. The objective is to help blood banks identify potential repeat donors and improve donation campaign planning through data-driven insights.

📌 Project Objectives
Perform Exploratory Data Analysis (EDA) on blood donation data.
Clean and preprocess the dataset.
Build and compare multiple machine learning classification models.
Evaluate model performance using standard metrics.
Select the best-performing model for predicting repeat blood donations.

✅ Results

After comparing multiple machine learning models, Logistic Regression was selected as the final model due to its balanced performance, interpretability, and strong generalization capability for predicting repeat blood donors.
